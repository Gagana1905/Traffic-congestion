## How to Run the Project

1. Import the Langflow workflow .json files from the flows/ directory
2. Configure IBM watsonx.ai API credentials to enable Granite foundation model access
3. Execute the traffic analysis workflow within Langflow
4. Interact with the system via a chat interface to obtain explainable traffic congestion insights

## Disclaimer

This project is developed for educational and demonstration purposes to showcase the use of IBM watsonx.ai, Granite models, and agentic AI workflows.
It does not provide real-time traffic control or operational decision-making in live city environments.
